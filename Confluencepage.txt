1️⃣ JIRA TICKET 1 – Line Manager Data Population
Title
User Recertification – Populate Line Manager Details in USER_TAB (Production)
Description
This task involves populating and validating Line Manager details in the USER_TAB table for users participating in the User Recertification process.
The Line Manager information (LM_ID, LM_NAME, LM_MAIL) will be populated using the application utility developed for Active Directory lookup.
The data population is required before triggering the User Recertification email workflow so that the correct Line Managers receive the recertification emails.
Scope
Populate Line Manager details in USER_TAB.
Validate LM mapping for selected users.
Ensure no null or incorrect mappings.
Perform validation queries after execution.
Acceptance Criteria
USER_TAB contains correct LM_ID, LM_NAME, LM_MAIL values.
Records verified for selected users.
No errors in execution logs.
