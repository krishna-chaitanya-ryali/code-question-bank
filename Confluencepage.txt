Ticket 1 — Analysis & Solution Design
Writing
Title: RAP Enhancement – Analysis and Solution Design for Access Request Form & Location Menu Visibility
Description: Perform detailed analysis for the enhancement request related to managing visibility of RMM locations in the RAP application.
Scope of Work: • Review current logic where the ACTIVE flag in RAP_RMM_GROUP controls visibility in both the RAP menu and Access Request form.
• Analyze the requirement to support standalone RMM locations.
• Identify design gaps in the current implementation.
• Prepare the proposed technical solution.
Proposed Design: Introduce two new columns in RAP_RMM_GROUP:
SHOW_IN_MENU
SHOW_IN_ACCESS_REQ
These columns will allow independent control of visibility in the menu and the access request form.
Deliverables: • Confluence documentation including problem statement, current logic, and proposed design.
• Discussion with stakeholders including business, ETL team, and Scrum Master.
Acceptance Criteria: • Confluence documentation completed.
• Solution design reviewed with stakeholders.
• Implementation approach finalized.
